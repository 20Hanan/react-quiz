# 🧠 React Quiz App

An interactive quiz application built with React that allows users to test their knowledge through multiple-choice questions.

The application presents questions one by one, tracks the user’s progress, calculates the score, and displays the final results at the end of the quiz.

This project focuses on practicing **advanced React concepts such as state management, reducers, and handling asynchronous data**.

---

## 🚀 Features

* ❓ **Multiple Choice Questions**
  Users can answer quiz questions by selecting one of several options.

* ⏱️ **Quiz Progress Tracking**
  The app keeps track of the current question and quiz progress.

* 🧮 **Score Calculation**
  The score updates dynamically as users answer questions.

* 📊 **Final Result Summary**
  Displays the final score once the quiz is completed.

* 🔄 **Restart Quiz**
  Users can restart the quiz and try again.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/20Hanan/react-quiz.git
```

Navigate to the project folder:

```bash
cd react-quiz
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

The application will run on:

```
http://localhost:3000
```
